Selamat Datang

Halo! Ini adalah contoh kerangka website dasar. Website ini dibuat hanya menggunakan satu buah file HTML. Pengaturan desain seperti warna, margin, dan ukuran teks dilakukan dengan CSS yang disisipkan langsung di dalam tag <style>.

Tentang Pembuatan Web

Membuat website dengan HTML murni adalah langkah pertama yang paling penting bagi pemula. Beberapa tag dasar (semantik) yang digunakan pada website ini antara lain:

  • <header>: Untuk kepala atau judul halaman.
  • <nav>: Untuk membuat menu navigasi.
  • <main>: Untuk membungkus konten utama.
  • <section>: Untuk membagi konten menjadi beberapa bagian.
  • <footer>: Untuk catatan kaki halaman.
